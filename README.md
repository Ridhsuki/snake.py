
# Snake.py

Simple Snake game built with Python and Tkinter. Demonstrates basic Python concepts like loops, keyboard input, and simple game logic.

---

## How to play

1. Clone the repository:
   ```
   git clone https://github.com/Ridhsuki/snake.py.git
   ```

2. Enter the folder:
   ```
   cd snake.py
   ```

3. Run the game:
   ```
   python main.py
   ```

4. Use the arrow keys to move the snake.  
Eat food to grow and avoid walls or your own tail.  
Press **SPACE** to restart after game over.
